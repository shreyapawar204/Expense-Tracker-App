# 📱 Personal Expense Tracker with Analytics

## 👩‍💻 Developed By

* Shreya Sayaji Pawar

## 🎯 Aim

To develop a mobile application that helps users track daily expenses, manage budgets, and analyze spending patterns.

## 📌 Features

* Add & manage income and expenses
* Categorize expenses (Food, Travel, Shopping, etc.)
* Budget tracking with alerts
* Charts (Pie & Bar) for analytics
* Secure login system

## 🛠️ Technologies Used

* Java (Android)
* SQLite
* XML
* MPAndroidChart

## 📊 Description

This app helps users manage personal finances by recording expenses and providing analytics through visual charts.

## 🚀 How to Run

1. Open in Android Studio
2. Run on emulator or mobile
